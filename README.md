# Live Three or Die Trying

This project analyzes whether the NBA three-point premium still exists.

The article uses team offensive data, three-point attempt rate, true shooting percentage, free throw rate, offensive rating, and cluster modeling to compare modern NBA offensive styles.

## Project Overview

The three-point shot is still valuable, but the edge is no longer automatic. This project looks at whether teams that take more threes are actually more efficient, or whether the modern premium depends more on context, balance, foul pressure, and offensive quality.

The analysis uses clustering to group NBA teams by offensive profile and then compares those groups across shooting volume, efficiency, free throw pressure, and offensive rating.

## Files

- `LiveThreeOrDieTrying.Rmd`: main R Markdown article
- `index.html`: knitted article for GitHub Pages
- `team_advanced.csv`: team advanced statistics used in the analysis
